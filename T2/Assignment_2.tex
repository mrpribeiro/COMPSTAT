\section{Dataset description}\label{dataset-description}

The original data was acquired from \textbf{critically-ill patients}
during hospital care at the Beth Israel Deaconess Medical Centre
(Boston, MA, USA). \textbf{Two annotators} manually annotated individual
breaths in each recording using the impedance respiratory signal. The
\textbf{53 recordings} within the dataset, each of \textbf{8-minute}
duration, each contain:

\begin{itemize}
\tightlist
\item
  Physiological signals include the photoplethysmogram \textbf{(PPG)},
  impedance respiratory signal, and electrocardiogram (\textbf{ECG}).
  These are sampled at \textbf{125 Hz}.
\item
  Physiological parameters, such as the heart rate (\textbf{HR}),
  respiratory rate (\textbf{RESP}), and blood oxygen saturation level
  (\textbf{SpO2}). These are sampled at 1 Hz.
\item
  Fixed parameters, such as \textbf{age} and \textbf{gender}
\item
  \textbf{Manual annotations of breaths}
\end{itemize}

\section{Exercise}\label{exercise}

Using all the files available in
\url{https://physionet.org/content/bidmc/1.0.0/}:

\begin{enumerate}
\def\labelenumi{\arabic{enumi}.}
\tightlist
\item
  Create an \textbf{object} containing the \textbf{mean HR and age} for
  \textbf{each subject}.
\item
  \textbf{Plot the histogram} of all the \textbf{HR means} with the
  respective \textbf{density curve}.
\item
  Apply the \textbf{expectation-maximization algorithm with 2 and 3
  latent classes}.
\item
  \textbf{Plot the densities} for \textbf{each} of the \textbf{latent
  classes}.
\item
  \textbf{Plot} the \textbf{mean} HR as a \textbf{function} of each
  subject's \textbf{age} in which \textbf{each point} has a
  \textbf{color} corresponding to the \textbf{class to which it
  belongs}.
\item
  \textbf{Generate 1000 data points} at random according to a
  \textbf{single distribution fitted to the original data}.
\item
  \textbf{Generate 1000 data points} at random according to a
  \textbf{mixture of distributions} fitted using \textbf{one of the
  previous EM} computed in 3.
\end{enumerate}

The submitted code file should include \textbf{comments} to improve
readability.

\href{\%5BNotes:\%5D(Notes:)\%7B.uri\%7D}{{[}Notes:\textbackslash\textbackslash{]}(Notes:)\{.uri\}}
- the filenames vary between bidmc\_01 to bidmc\_53\\
- the \textbf{bidmc\_XX\_Numerics.csv} contains four columns with the
physiological parameters, such as the heart rate (\textbf{HR}),
respiratory rate (RESP), and blood oxygen saturation level (SpO2). These
are sampled at 1 Hz.\\
- the \textbf{bidmc\_XX\_Fix.txt} contains the values of Fixed
variables, such as \textbf{age} and gender.\\
- the link to download the files are in the following format:\\
``\url{https://physionet.org/files/bidmc/1.0.0/bidmc_csv/bidmc_01_Numerics.csv?download}''\strut \\
and\\
``\url{https://physionet.org/files/bidmc/1.0.0/bidmc_csv/bidmc_01_Fix.txt}''\strut \\
- do not also forget to use the \textbf{functions density and EM}
already \textbf{available in R}.
